# Finkita

Financial Tracker App

## How to run

### Prerequisite

- go ^1.15
- docker
- docker-compose

run service

```bash
make run
```

run service in development mode (hot reload with `air`)

```bash
make dev-air
```

run test

```bash
make test
```

build service

```bash
make build
```

run scraper

```bash
make run-scraper
```

build scraper

```bash
make build-scraper
```
